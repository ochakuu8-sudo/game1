# Mining Pinball

Mining Pinball は、ピンボールのボールを地面ブロックに当てて掘り進める採掘アクションゲームです。
時間経過で自動収益は発生せず、プレイヤーが直接ブロックを壊した瞬間だけお金を獲得します。
一定の深さまで掘ると盤面がスクロールし、さらに深い地層と高価な鉱石が出現します。

## ゲームの流れ

1. ボールを打ち出す
2. 地面ブロックに当てて耐久値を削る
3. ブロック破壊で穴を広げる
4. 鉱石破壊でお金を獲得
5. 一定ラインまで掘ると盤面が前進（深度上昇）
6. 深い地層ほど価値の高い鉱石が出現

## 操作方法

- 左フリッパー: `←` / `A`
- 右フリッパー: `→` / `D`
- 打ち出し（長押しチャージ）: `Space`
- 採掘力アップグレード: `U`
- リスタート: `R`
