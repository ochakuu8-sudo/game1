# KAIJU PINBALL

怪獣が街を壊すピンボールゲーム。ブラウザ / スマホ対応、itch.io 公開用。

- 建物にはHPが表示され、ボールを当てると減っていく
- 建物のHPが0になると倒壊し、逃げ惑う人間が出現する
- 人間をボールで潰していくと一定数ごとにマルチボール発生
- 建物破壊のたびにパワーアップ（フリッパー強化・キャッチ範囲UP・スコア倍率・キック力UP・エクストラボール）が積み重なっていく、時間制限のないスコアアタック

## 技術構成

- **PixiJS (WebGL)** — 起動時に建物・人間・ボール・パーティクル・数字を1枚のテクスチャに焼き込み（`src/core/atlas.ts`）、共有アトラスにすることで大量の人間・パーティクルが出てもドローコールが増えないようにしている。人間やデブリ・スパークは `ParticleContainer` 上の固定プールで使い回し、スマホでも軽く動くようにしている
- **Matter.js** — フリッパー・バンパー化した建物・壁などのピンボール物理
- 人間の徘徊/逃走AIは物理エンジンに乗せず、軽量な独自ステップ処理

## 開発

```bash
npm install
npm run dev      # 開発サーバー
npm run build    # dist/ に本番ビルド
npm run zip      # ビルドして release/kaiju-pinball-web.zip を作成 (itch.ioにそのままアップロード可)
```

## 操作

- 画面左半分タップ/長押し = 左フリッパー、右半分 = 右フリッパー
- ボールが無いときは画面タップで発射
- キーボード: ←/Z = 左フリッパー、→/M = 右フリッパー、Space = 発射（PC確認用）
